# Blag Platform

A Modern MERN Stack Blog Application.

## Technology Stack

### Client

- **Framework**: React with Vite
- **State Management**: Redux Toolkit
- **Styling**: Tailwind CSS, Shadcn/UI
- **Editor**: TipTap (Rich Text Editor)
- **HTTP Client**: Axios

### Server

- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB (Mongoose)
- **Authentication**: JWT (JSON Web Tokens)
- **Image Storage**: Cloudinary

## Features

- **User Authentication**: Secure Sign up, Login, and Logout functionality.
- **Blog Management**: Create, Read, Update, and Delete (CRUD) blog posts.
- **Rich Text Editor**: Advanced text editing capabilities using TipTap.
- **Image Uploads**: Seamless image uploading via Cloudinary.
- **Interactive Comments**: Engage with posts through a commenting system.
- **Responsive Design**: Fully responsive UI tailored for mobile and desktop.
- **Theming**: Support for user-preferred themes (Dark/Light mode).

## Installation

### Prerequisites

Ensure you have the following installed:

- Node.js
- MongoDB

### Client Dependencies

Navigate to the `client` directory and install the necessary packages:

```bash
cd client
npm install
```

### Server Dependencies

Navigate to the `server` directory and install the necessary packages:

```bash
cd server
npm install
```
